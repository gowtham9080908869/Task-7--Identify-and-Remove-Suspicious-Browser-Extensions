# Task-7--Identify-and-Remove-Suspicious-Browser-Extensions

 Overview
This repository documents the results of a security audit performed on browser extensions installed in Google Chrome and Opera. The primary goal of this review was to evaluate installed extensions, assess their permissions, detect any suspicious or harmful activity, and record the measures taken to strengthen browser security and performance.

Objectives

- Conduct a full review of installed extensions
- Flag unused or potentially risky extensions
- Assess threats posed by malicious extensions
- Record remediation steps and final security posture
